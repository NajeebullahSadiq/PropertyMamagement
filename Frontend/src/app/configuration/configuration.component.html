راهنمای تنظیمات

از بخش لیست کاربران می‌توانید کاربران جدید را مشاهده و مدیریت کنید
از بخش مدیریت صلاحیت‌ها می‌توانید صلاحیت‌های هر کاربر را تعیین کنید
از بخش صلاحیت‌های نقش‌ها می‌توانید نقش‌های سیستم را مدیریت کنید
از بخش مدیریت ولسوالی‌ها می‌توانید ولسوالی‌ها را فعال کنید
از بخش محل فعالیت عریضه‌نویس می‌توانید محل فعالیت‌ها را مدیریت کنید
از بخش لاگ فعالیت‌ها می‌توانید تمام فعالیت‌های سیستم را بررسی کنید
